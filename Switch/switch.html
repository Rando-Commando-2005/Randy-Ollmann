Button Toggle

This button will change the state of an element back and forth on each click.

Button Cycle

This button will cycle the state of an element through several options.

Switch

This will change the state of an element only if the correct button is pressed.

SR nor Latch

These will change the state of an element only if the correct switch is activated.

Funny Button

This button only works if you press the redonkulous button

Redonkulous Button

This button only works if you press the funny button

Exclusive NOR Gate

This only shows red if a or b is not pressed but doesn't show red if both aren't pressed.

Smooth Button

This button changes smoothly.
